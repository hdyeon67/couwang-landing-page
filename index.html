쿠폰·기프티콘 만료일 관리 앱

흩어진 쿠폰과 기프티콘, 쿠왕이 한곳에 모아드립니다.

쿠왕은 쿠폰과 기프티콘을 등록하고, 만료일 기준으로 확인하며, 사용 시 필요한 바코드와 멤버십까지 함께 관리할 수 있도록 돕는 Flutter 기반 모바일 앱입니다.

Android와 iOS에서 사용할 수 있도록 기획·개발·출시한 생활 편의 앱 프로젝트입니다.

Google Play에서 쿠왕 다운로드 App Store에서 쿠왕 다운로드 주요 기능 보기
  • 쿠폰 등록
  • 만료일 관리
  • 바코드 확인
  • 멤버십 관리
  • 만료 알림

Problem

쿠폰은 받았는데, 정작 사용할 때는 놓치기 쉽습니다.

쿠폰과 기프티콘은 카카오톡, 문자, 이미지, 이벤트 페이지 등 여러 경로로 받게 됩니다. 하지만 저장 위치가 흩어져 있거나 만료일을 기억하지 못하면, 사용 가능한 혜택도 그대로 지나치기 쉽습니다.

01

어디에 저장했는지 잊어버림

받은 쿠폰이 카카오톡, 문자, 갤러리 등에 흩어져 있어 한눈에 확인하기 어렵습니다.

02

만료일을 놓침

사용하려고 봤을 때 이미 만료된 경우가 생기고, 쿠폰을 실제 혜택으로 연결하지 못합니다.

03

사용할 때 다시 찾기 번거로움

매장에서 바코드, 브랜드명, 유효기간, 멤버십을 각각 찾아야 해 사용 흐름이 불편합니다.

Solution

쿠왕은 쿠폰을 실제로 사용할 수 있도록 관리 흐름을 정리했습니다.

쿠왕은 단순히 쿠폰 이미지를 저장하는 앱이 아니라, 쿠폰 등록부터 만료일 확인, 바코드 사용, 멤버십 관리, 알림 설정까지 이어지는 흐름을 하나의 앱 안에서 관리할 수 있도록 구성했습니다.

01

한곳에 모아 관리

쿠폰과 기프티콘을 앱에 등록하고 목록에서 한눈에 확인할 수 있습니다.

02

만료일 기준으로 확인

D-Day 기준으로 남은 기간을 확인하고, 임박한 쿠폰을 우선적으로 볼 수 있습니다.

03

사용 순간까지 고려

상세 화면에서 바코드와 QR을 확인하고, 멤버십 바코드까지 함께 관리할 수 있습니다.

Features

쿠왕의 주요 기능

쿠폰을 등록하고, 확인하고, 사용하는 과정을 모바일 환경에 맞게 단순하게 구성했습니다.

쿠폰·기프티콘 등록

이미지나 교환 코드를 기반으로 쿠폰 정보를 등록할 수 있습니다.

만료일 기준 목록 관리

등록한 쿠폰을 사용 가능, 사용 완료, 만료 상태로 구분하고 만료일 기준으로 확인할 수 있습니다.

바코드/QR 확인

매장에서 사용할 수 있도록 상세 화면에서 바코드와 QR 정보를 확인할 수 있습니다.

이미지 기반 정보 추출

쿠폰 이미지에서 필요한 정보를 추출해 등록 과정을 줄일 수 있는 흐름을 구성했습니다.

멤버십 바코드 관리

쿠폰 사용 시 함께 필요한 멤버십 바코드를 앱 안에서 관리할 수 있습니다.

만료 알림 관리

만료 전 알림과 최근 알림 확인 기능을 통해 쿠폰 사용 시점을 놓치지 않도록 돕습니다.

App Screens

실제 사용 흐름에 맞춘 화면 구성

쿠폰을 등록하고, 목록에서 확인하고, 상세 화면에서 사용하는 흐름이 자연스럽게 이어지도록 화면을 구성했습니다.

쿠폰 목록 화면

보유한 쿠폰을 상태별로 확인하고, 만료일 기준으로 관리할 수 있습니다.

쿠폰 상세 화면

쿠폰 정보와 바코드/QR을 확인하고, 사용 완료 처리까지 진행할 수 있습니다.

쿠폰 등록 화면

이미지 선택, 교환 코드, 브랜드, 유효기간 등 필요한 정보를 입력할 수 있습니다.

멤버십 화면

자주 사용하는 멤버십 바코드를 등록해 쿠폰 사용 흐름과 함께 관리할 수 있습니다.

알림 화면

만료 전 알림과 최근 알림을 확인해 쿠폰 사용 시점을 놓치지 않도록 돕습니다.

Project Story

기획부터 개발, 출시까지 직접 진행한 프로젝트
